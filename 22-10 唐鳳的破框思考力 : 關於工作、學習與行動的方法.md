#### Part 1 我這樣思考――共好、共創、共享
###### Chapter 1 建構多元的知識系統：我自學，但我很少在家
1. 自學路上，讓自己不孤單最重要
   + 視域融合
   + 不把事情想通的擔子放在個人身上。
2. 從線性教育的枷鎖中走出來
   + 台中尋找一個聚會的場所。
   + 清楚易懂的見解成了一種力量
3. 知識必須透過討論，才能產生價值
   + 用拼圖概念取背誦單一標準答案的方式
###### Chapter2 比世界更大的世界：閱讀開啟深度思考
1. 科幻小說是一種思考未來的訓練
   + 艾西莫夫、班克斯
   + 新技術開發可加入倫理
   + cooike的技術案例
2. 姜峯楠的科幻小說，融合科幻與哲學思考
   + 你一生的預言(異星入境)、呼吸。天註定。
   + 共同經驗就像是準心，用準心來校準，才能學到東西
3. AI不是一開始就聰明，它需要一段熟成過程
   + 用共存的方式，整個社會與AI慢慢成熟，非極權方式。
   + 維根斯坦(哲學的任務，就是給蒼蠅指出捕蠅瓶的出口)、喬伊斯
4. 不同經典可以提供不同價值觀，避免陷入單一價值窠臼中
   + 自己有才能沒有用，要有所貢獻，不是由自己來認定。
   + 芬尼根守靈夜
   + 一個人的程式能力取決與語言的能力。
   + 張三豐的共創精神，勝過於比武勝敗。

###### Chapter3 以共好取代競爭：與他人共同成就彼此
1. 分享，價值就不會消失
   + 因心臟疾病，擔心沒分享就會白想。
2. 把想法放在平台上，讓價值擴散出去
 
#### Part 2 我這樣工作――跨域協同
###### Chapter 4 人生兩次工作大游牧：開放，全世界都是你的
1. 共好：帶著探索的好奇心，到世界各地取經
   + 1997年，16歲開源運動到美國取經
   + 2005年，24歲Perl壯遊計畫，轉性
2. 網路時代知識無法囤積，一定有人會找到解方分享出去
   + publish or perish
3. 從擁有技術者，到成為給予者
   + 打破從屬關係
   + 你的理念寫得越清楚，越多人加入。
   + 2008年 socialtext
4. 共創：體驗遠距工作帶來的職場新文化
5. 把每個員工當成創業夥伴對待
   + 當個大人
6. 共感：善用科技工具，建立工作流程，營造共同空間
   + 如何確保員工的能動性不會在組織中被消磨掉-提供更廣闊，引道到有創造能量的方式。
   + 2016入閣
###### Chapter 5 成為時間的主人：主導自己的人生
1. 善用新服務幫你省時
   + 紀錄夢境，
   + 享廚好食
2. 自己動手料理既健康，也是一種放鬆
3. 成為時間主人的番茄鐘工作法
   + 一天只回兩次LINE
   + 工作25分，休息5分。
4. 用觸控筆滑螢幕，避免過度使用3C產品
5. 清除臉書介面，不被雜訊干擾
6. 主動掌握時間結構，才能拿回時間主導權
   + 阿德勒:除非你運用紀錄，讓你的心靈自由，不然無法達到真實的自由
###### Chapter 6 維持專注力與好情緒的必要：GTD 工作法和精神按摩法
1. 給自己緩衝時間的工作節奏
   + 每週三五遠距上班，原因是面對面很容易被實體面對打擾。
   + 每個人都有自己的時間結構，要懂得尊重。
   + GTD工作法:
     + 收件視角
     + 回顧視角
2. 精神也需要按摩，才能強化玻璃心
   + 例如按摩到痛點沒有被揉開，透過睡覺來回覆。
   + 擁抱小白要在心情平復前
3. 面對酸民，只須回應有建設性的話語
   + 比暴力更可怕的是冷漠
###### Chapter 7 睡眠記憶法：善用大腦的運作
1. 讓自己每天歸零，才可能裝載新事物
   + 睡前四件事
     + 關機
     + 待辦清空
     + 閱讀大量知識文件
     + 打坐20分鐘
2. 睡前正確的閱讀，才能留下記憶
   + 不打斷作者
3. 從練習不打斷別人說話開始
   + 多少時間就是晚上的多少時間
   + 你容忍的能力有多大表示你能解決多複雜的問題
   + 補眠的技巧:
     + 補眠要在另外的空間
     + 補眠前喝咖啡
4. 以關鍵字記憶書中概念
   + 工作記憶，紀錄需要的關鍵字。
   + 一起床要先思考要做什麼，不要立刻踏入另外一個世界，沒有給自己一個緩衝期

###### Chapter 8 領導不是管理：團隊需要的是水平協調者
1. 不下命令，讓成員自訂工作目標
   + 團隊必須多元，一個部會派一個。
   + 水平管理，自行決定OKR，而非有上而下的KPI。
2. 公開工作內容，協助彼此完成目標
   + 大聲工作法:看板管理法，讓大家知道你在做什麼。
   + 主管的職責就是看盤，讓大家能整合或碰撞出新的火花。
   + 領導是讓成員保有主動解決的能力。
3. 領導是讓成員保有主動解決問題的動力
   + 領導者要勇於拋開面子問題，鼓勵用共創的方式，企業才能永續
###### Chapter 9 打破平行時空的開會法：開門造車，你行你來
1. 達成「粗略的共識」就去執行
   + 焦點討論法
     + 觀察事實與外在現況
     + 說出感覺與聯想
     + 尋找意義
     + 做出決策採取行動
2. 引導大家進入同一個思想脈絡
   + 動態引導
   + 培力最靠近痛點的人，儘量創造與議題相關的共同經驗
3. 不用一步到位，足夠好就去執行
   + 夠好就好，最好是夠好的敵人
4. 開放參與，能力才能傳承
   + 越是把想法形成的過程，放在自己這一代的人，越難二代接班，因為年輕人不知道為何做這樣的決定。
   + 急著把問題解決，其他人就學不到東西。
5. 每次開會都要做逐字稿紀錄
   + 你有這麼強烈的感受，是不是觀察到一些事實，可否分享
6. 讓非專業者擔任會議主席，傾聽不同聲音
   + 主持團隊要站在一般民眾的立場。
7. 在不同專業間，找到共同價值
 
#### Part 3 我這樣學習――學習就是自我啟蒙
###### Chapter10 人非機器，學習讓自己成為「無用之人」
1. 學習不該只強調有用，應發自內心興趣
   + 當人被物化時，容易被侷限在每一功能，當被取代時就容易有挫折感，因此要發自自己興趣才有價值。
2. 不強調人具備某種特定用途
   + 有了興趣，了解社會需求產生共同價值，對社會才有親近性。
   + 比馬龍效益:13歲認為自己是成年人了
3. 透過集體共創的智慧，探索新知識
   + 後缺稀時代強調共感與共創。
4. 知識性事物適合線上學習；操作性事物則適合線下共聚一堂
5. 營造一個可以犯錯的空間
   + 英文的語法能意會比較重要。
