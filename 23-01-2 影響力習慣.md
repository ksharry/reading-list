## 第一部 影響力習慣
#### 第一章 影響力成員
1. 影響力成員
   + 高影響力貢獻者:在工作上展現超群價值和影響力的人 
   + 一般貢獻者:聰明、有能力,工作表現不錯(但不是特別好)
   + 低貢獻者:聰明、有能力,但表現低於他們的能力水準的人
2. 了解影響力成員
   + 影響力成員帶者機會眼鏡
   + 增添價值的機會
   + 影響力成員對於不確定性有不同的反應
     + 做需要做的事
     + 站出來，退回去
     + 堅持到底
     + 尋求回饋意見並調整
     + 讓工作變輕鬆
   + 影響力成員敢採用不成文規定
   + 影響力帶來投資
   + 建立價值
   + 連鎖反應
   + 被忽略的成員
3. 倍增你的影響力
   + 教戰手冊
   + 成員
   + 建立高影響力心態
