## Part I 破解7大產業財報
#### 第1章 電子組裝業財報解析
1. 產業分類
   + 原件(鋁箔)
   + 機構建(主機板)
   + 最終產品(筆電)
2. 電子毛利及低，營收數字是獲利關鍵
3. 新電子五哥:鴻海、和碩、廣達、仁寶、緯創
4. OEM/ODM
5. 國際大廠5大能力
   + 關鍵零件的提供或建議能力
   + 全球組裝能力
   + 全球交貨能力
   + 全球維修能力
   + 全球客戶服務能力
6. 資產負債五大特色
   + 應收與存貨及高
   + 現金龐大
   + 高流動資產
   + 權益法偷資金額大
   + 負債比率偏高
7. 損益表五大特色
   + 營收驚人
   + 毛利偏低
   + 營業費用極低
   + 投資理財收益多
   + 營利表現尚可
#### 第2章 IC設計業財報解析
1. 產業分類
   + IC設計
   + 晶圓代工
   + 封測業
2. IC設計的商業模式
   + 發展智財
   + 設計自有產品
   + 設計客製化晶片
3. 資產負債五大特色
   + 不動產、廠房及設備比重低
   + 現金與長短期投資多
   + 存貨是罩門所在
   + 無形資產占比高
   + 負債比率低
4. 損益表五大特色
   + 毛利率高
   + 研發費用高
   + 獲利變動大

#### 第3章 實體通路業財報解析
1. 實體通路的四種經營模式
   + 便利商店
   + 超市
   + 量販店
   + 百貨公司
   + 特定商品連鎖定
2. 經營模式
   + 加盟模式1:7-11、抽成
   + 加盟模式2:加盟者的收入
   + 直營店模式
   + 專櫃模式
3. 便利商店
   + 便利商店優勢
     + 優勢1:通路規模
     + 優勢2:服務密度
     + 優勢3:管理力度
     + 優勢4:不斷創新
     + 優勢5:完整體系
  +  資產負債表特性
     + 無應收票據與帳款
     + 存貨周轉率20-30天
     + 使用權資產與租任負債金額驚人
     + 其他應付款偏高
     + 流動比率偏低:正常不可低於120%
     + 負債比率高:不宜高過65%
  +  損益表特性
     + 毛利率超級穩定
     + 極高的推銷費用率
     + 很低的營業利益率
     + 投資利益高
     + 稅後淨利率不高但獲利驚人
4. 特定商品連鎖業
  +  資產負債表特性
     + 無應收票據與帳款
     + 存貨金額及比重很高
     + 使用權資產與租任負債金額驚人
     + 流動比率高低，依產業而定
     + 負債比率高依商業模式而定
  +  損益表特性
     + 穩定毛利率
     + 穩定推銷費用率
     + 很低的營業利益率
#### 第4章 連鎖餐飲業財報解析
1. 餐飲業經營議題
   +  價格
   +  顧客忠誠度
   +  翻桌率
   +  經營成本
   +  人員流動率
   +  擴大規模
2. 大型餐飲業商業模式
   + 直營店模式
   + 加盟店模式
   + 兼採直營與加盟
3. 直營店模式
   + 資產負債表特性
     + 不多的應收
     + 存貨週轉天數不高於一個月
     + 資產占比與製造業類似
     + 使用權與租任負債大
     + 流動比率不宜過低
     + 負債比率高
   + 損益表特性
     + 毛利率依價格而定
     + 推銷費用率依精緻度而定
     + 管理費用很高
     + 固定成本高，營收驟降易虧損
3. 加盟店模式
   + 資產負債表特性
     + 不多的應收
     + 存貨不宜過高
     + 資產依實財複雜度而定
     + 使用權資產沒有
     + 營運需要有其他應收款
     + 營運需要產生預付款
     + 流動資產比率不宜過低
     + 負債比率遠比直營店模式低
   + 損益表特性
     + 反推事的毛利率
     + 推銷費用率差異相當大
     + 管理費用不低
     + 營業利益率差異大
#### 第5章 生技醫療業財報解析
1. 五大類
   + 藥品研發
   + 藥品製造
   + 醫財產業
   + 藥品與醫材通路
   + 保健食品
2. 藥廠商業模式
   + 研發委外或購買，更有效率且財報好看
   + 無形資產占比高
3. 台灣藥品研發業:浩鼎、中裕
   + 兩階段
     + 臨床前研究
     + 人體臨床實驗，分三期
   + 商業模式
     + 大多只做研發，不做生產與銷售
       + 缺乏資金
       + 缺乏銷售通路
     + 階段性的授權收入模式
       + 簽約金
       + 間段授權金
       + 授權共同開發
       + 案銷售淨額的依定比率收取權利金
   + 藥品研發業的財務報表
     + 看現今還能燒多久
     + 留意新藥專利的利多出盡
     + 研發案少，成敗賭住大
2. 藥品製造業:永信(學名藥)、中化、東洋、神隆(原料藥)
   + 資產負債表
     + 資產
     + 使用權資產
     + 應收帳款
   + 損益表
     + 毛利率高
     + 推銷費用高
3. 醫材業
   +類似製造業
   + 資產負債表
     + 存貨
     + 應收帳款

#### 第6章 銀行業財報解析
1. 銀行業商業模式
   + 資產負債表
     + 負債面
       + 透過損益按公允價值衡量之金融資產-會被認列EPS-最透明
       + 透過其他綜合損益按公允價值衡量之金融資產-處份時認列EPS-可操作損益穩定
       + 按攤銷後成本衡量之債務工具投資-處份後認到保留盈餘-最穩健
     + 資產面
     + 放款及存放同業面
     + 投資與市場操作面
   + 損益表
     + 利息淨收益
     + 手續費淨收益
   + 銀行業經營績效評估
     + 成本收益比
     + 逾放比及逾期放款覆蓋率
     + EPS
     + ROE/ROA
     + 投資未實現收益

## Part II.讀懂財報眉角
#### 第8章 讀財報˙起手式
1. 消息面重要，還是財報面重要？
   + 消息面
2. 財報總是厚厚一本，如何更有效率的解讀？
   + 損益表美女/資產負債表內在
   + 三大報表5,6,8
   + 9.6,7,8,9,1,0,11,13,14
   + 或有負債:官司賠償
3. 如何解讀現金流量表中的營業活動、投資活動及籌資活動？
   + 稅前淨利+折舊+攤銷-所得稅 接近營業活動現金數就表示專注本業。
4. 何謂「結構性獲利能力」？
   + 收入-成本-相關費用與稅捐後，能為其投入的資源，轉得滿意的長期利潤
     + 例如全聯、壓低價格，取得規模經濟
   + 綜上，營收、毛利率、費用率、負債比率極ROE表現才能評斷
     + 鴻海ROE低，但是透過降低股東權益來提高負債比非好事，可能是其他四間太高的負債比。
     + 聯發科財務長會理財，導致負債比低，建議要將多餘的現金退還給股東。
