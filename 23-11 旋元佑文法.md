## Part 1單句
#### 第1章 基本句型
1. 句型
   + 兩種元素
     + SV
   + 三種元素
     + SVC
     + SVO
   + 四種元素
     + SVOC
     + SVOO
2 重點
  + O所謂「補語」,就是交代「是什麼」。
  + 主詞補語Sv+c的方式是把動詞換成be動詞,句子話就是補語,不然就是受詞。
  + 檢驗受詞補語〈S+v+0+C)的方式則在受詞和第四個元素之間加個be 動詞,意思大致相同的話,後面就是補語,否則就是第二個受詞。
#### 第2章 名詞片語
1. 可數名詞
   + 固定形狀 boy
   + 度量單位 feet
2. 不可數名詞
   + 物質名詞 paper
   + 抽象名詞 wisdom
   + 動名詞 Smoking
   + 專有名詞 London
   + 複合名詞
     + The dishwasher 名詞當形容詞
     + The best pole-vaulter 連接詞
     + The flower shop 名詞當形容詞
   + 複合名詞的複數
     + Three gift shops
     + two-brothers-in-law
     + push-ups
3. 名詞片語的副詞
4. 名詞片語的形容詞/可以兩個以上
5. 名詞片語的省略 the rich
6. 限定詞的種類 
   + 前位限定詞 all/such
   + 中位限定詞 冠詞/所有格;
   + 後衛限定詞 多少的/three
   + 注意排他性
7. 名詞片語中的名詞片語
8. 名詞與限定詞的一致姓 
   + many much some
   + few/fewer 搭配可數
   + little/less 搭配不可數
   + number 搭配可數
   + among 搭配不可數
9. 數字的寫法
10. 冠詞
    + 不定冠詞/one的弱化用a，反之用零冠詞
    + 零冠詞/省略The
    + 定冠詞/that的弱化用the
11. 專有名詞/不加冠詞
12. 假專有名詞/The Times
13. 專有名詞普通化 a Harry 當作想像中的Harry這個人
14. 普通名詞專有化/同位語
15. 普通名詞抽象化
16. 重點
    + 限定詞+形容詞+名詞
    + 名詞部分要注意的是可數、不可數。
    + 形容詞部分可以放入一個以上的形容詞(見第4章)。如果放入的是另一個名詞當作形容詞使用,就會形成複合名詞。
    + 限定詞中最常見的是冠詞和所有格,其中冠詞分為定冠詞(the)、不定冠詞(a/an)、零冠詞。定冠詞the重點在於「指示性」,零冠詞的重點則在於「不加限定」或「無限」。
    + 以上構成名詞片語的三個部分,只要符合條件、語意清楚,分別都有可能被省略。
#### 第3章 代名詞
1. 主格/受格
2. 所有格限定詞與所有格代名詞
3. 反身代名詞/副詞
   + 受詞:同主詞
   + 副詞:不一定同主詞
4. 常見的錯誤
   + 代名詞與主詞不一致。
   + 代名詞與先行詞單複數不一致。
   + 代名詞的性別考量/女權運動改she
   + 代名詞必須要有先行詞/do so
   + 代名詞只能有一個先行詞

#### 第5章 副詞
1. 副詞修飾的對象:動詞、形容詞、副詞
2. 副詞的位置:
   + 方法狀態副詞:修飾動詞怎麼作。
     + 儘量靠近要修飾的東西。
     + S + V :可前可後
     + S + V + C :
     + S + V + O :建議在動詞前
     + S + V + O + O :建議在動詞前
     + S + V + O + C
   + 頻率副詞
     + 修飾動作頻率，普通動詞前面，BE動詞與助動詞後面
   + 強調範圍的副詞
     + 修飾詞類不拘
     + 通常前面，例如only
   + 加強語氣的副詞
     + 修飾詞類不拘，例如badly
   + 程度副詞
     + 修飾詞類不拘，如拿掉會影響到語意
   + 連接副詞
     + 與連結詞不同，需要有逗點或分號斷開。
   + 分離副詞
     + 修飾整句或省略，例如theoretically
   + 介副詞
     + 後面沒有副詞，自己當副詞的介系詞。
   + 時間副詞與地方副詞
     + 修飾名詞
     + 修飾動詞
     + 作為補語
3. 副詞種類與位置比較
   + 練習不同例句
4. 形容詞與副詞的判斷
5. 重點
   + 副詞可以修飾的對象一般是動詞、形容詞、副詞,但有幾類副詞也可以修飾名詞。
   + 副詞的位置應盡量與修飾對象接近,而且對象要明確,避免一個飾多個對象都說得通,造成讀者難以確認句意。
   + 依照句型和語意要求,句子中的修飾語可能採用形容詞或副詞,要謹愼判斷。
#### 第6章 比較句法
1. 前言
   + 形容詞與副詞
   + as或than引導的副詞子句
2. 比較級變化
   + 單音節:er/est
   + 三音節以上:more/most
   + 兩音節，有標示詞類的字尾:more/most
   + 兩音節，無標示詞類的字尾:兩者都行
   + 不規則變化:good/better/best
3. 大於、小於、等於
   + 大於:省略主詞與動詞，用than作連結詞成副詞子句
   + 小於:not so ... as
   + 等於:as..as
4. 準關係代名詞
   + 比較子句與關係子句的差異
     + 是副詞子句
     + 修飾限定詞Many
     + 從屬連接詞
5. 定冠詞的判斷
   + 用名詞片語或補與位置判斷
6. that與those的使用
   + 單複數的判斷
7. 比較句法的平行對稱要求
   + 注意比較的重點，省略比較點，省略動詞，將主詞變代名詞
   + 其他比較中的平行名稱。
8. 比較句法省略的條件
9. 比較句的倒裝
10. 比較點的觀察:more than
11. 重點
    + 比較句法必須符合平行對稱的要求
    + 爲了避免内容累贅,可以省略副詞子句中重複的比較點、運用代名詞,但都必須以語意清楚為原則。
    + 為了增加清楚性,比較句法有時也可以倒装。
#### 第7章 介系詞

4. 空間介係詞

#### 第9章 動詞時態
1. 簡單式
   + 過去時間

     + 沒未來式，語氣助動詞/must/should/will/would/can/could/may/might
     + will和be going to
     + 來去動詞用現在式或現在進行式，因為已經排定/come/go/leave/arrive
     + 過去時間的未來

     + 名詞子句將問句簡化，副詞子句形容主句動詞
       + 了解文法規則
2. 完成式
   + 過去時間

#### 第10章 語態
1. 受詞拿出來當主詞，句子就要改寫成被動態
2. 基本句型的被動態
   + S + V + O -> S + V + C
   + S + V + O + O -> S + V + C
   + S + V + O + C -> S + V + C
3. 完成式的被動態
4. 進行式的被動態
5. 文法規則:表示人的感覺，要用被動態?
   + 如果意思是「令人感覺如何」,就是表達主動的形容詞,要用現在分詞(Ving)。
   + 如果意思是「覺得如何」,就是表蓬被動的形容詞,用過去分詞(Ven)。
6. 被動態必須師出有名
   + 通常使用主動態
   + 增加修飾效果
     + 連貫性 - 主詞相同
     + 清楚性 - 避免代名詞當主詞

#### 第12章 語氣
1. 直說語氣
   + 用在過去或現在完成
   + 未來確定用be going to/不確定用will
2. 條件語氣
   + 語氣助動詞 + 圓形動詞
   + should study
3. 假設語氣
   + 現在時間
     + If...,(then)...
       + 事實:現在式
       + 非事實:過去式+假設語氣
       + 條件副詞，BE動詞可以倒裝
     + wish...
       + 現在式 + 過去
     + It is time that...
   + 過去時間
     + 過去完成+過去簡單
     + 假設過去完成+過去完成
   + 未來時間
     + 未來式 + 未來式
     + 小整理
       + 不確定:語氣助動詞
       + 事前排定:當作事實用現在式
       + 時間副詞紫劇、條件副詞紫劇中:當作事實，用現在式
       + 未來無可能發生:用were to
       + 希望渺茫，但不排除萬一:用should
     + 混雜時間或語氣
4. 命令語氣
