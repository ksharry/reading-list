## Part 1單句
#### 第1章 基本句型
1. 句型
   + 兩種元素
     + SV
   + 三種元素
     + SVC
     + SVO
   + 四種元素
     + SVOC
     + SVOO
2 重點
  + O所謂「補語」,就是交代「是什麼」。
  + 主詞補語Sv+c的方式是把動詞換成be動詞,句子話就是補語,不然就是受詞。
  + 檢驗受詞補語〈S+v+0+C)的方式則在受詞和第四個元素之間加個be 動詞,意思大致相同的話,後面就是補語,否則就是第二個受詞。
#### 第2章 名詞片語
1. 可數名詞
   + 固定形狀 boy
   + 度量單位 feet
2. 不可數名詞
   + 物質名詞 paper
   + 抽象名詞 wisdom
   + 動名詞 Smoking
   + 專有名詞 London
   + 複合名詞
     + The dishwasher 名詞當形容詞
     + The best pole-vaulter 連接詞
     + The flower shop 名詞當形容詞
   + 複合名詞的複數
     + Three gift shops
     + two-brothers-in-law
     + push-ups
3. 名詞片語的副詞
4. 名詞片語的形容詞/可以兩個以上
5. 名詞片語的省略 the rich
6. 限定詞的種類 
   + 前位限定詞 all/such
   + 中位限定詞 冠詞/所有格;
   + 後衛限定詞 多少的/three
   + 注意排他性
7. 名詞片語中的名詞片語
8. 名詞與限定詞的一致姓 
   + many much some
   + few/fewer 搭配可數
   + little/less 搭配不可數
   + number 搭配可數
   + among 搭配不可數
9. 數字的寫法
10. 冠詞
    + 不定冠詞/one的弱化用a，反之用零冠詞
    + 零冠詞/省略The
    + 定冠詞/that的弱化用the
11. 專有名詞/不加冠詞
12. 假專有名詞/The Times
13. 專有名詞普通化 a Harry 當作想像中的Harry這個人
14. 普通名詞專有化/同位語
15. 普通名詞抽象化
16. 重點
    + 限定詞+形容詞+名詞
    + 名詞部分要注意的是可數、不可數。
    + 形容詞部分可以放入一個以上的形容詞(見第4章)。如果放入的是另一個名詞當作形容詞使用,就會形成複合名詞。
    + 限定詞中最常見的是冠詞和所有格,其中冠詞分為定冠詞(the)、不定冠詞(a/an)、零冠詞。定冠詞the重點在於「指示性」,零冠詞的重點則在於「不加限定」或「無限」。
    + 以上構成名詞片語的三個部分,只要符合條件、語意清楚,分別都有可能被省略。
#### 第3章 代名詞
1. 主格/受格
2. 所有格限定詞與所有格代名詞
3. 反身代名詞/副詞
   + 受詞:同主詞
   + 副詞:不一定同主詞
4. 常見的錯誤
   + 代名詞與主詞不一致。
   + 代名詞與先行詞單複數不一致。
   + 代名詞的性別考量/女權運動改she
   + 代名詞必須要有先行詞/do so
   + 代名詞只能有一個先行詞
#### 第4章 形容詞
1. 類型
   + 名詞片語(attributive adjectives)
     + 名詞轉形容詞使用-單數
     + 複合形容詞-改造單字
     + 複數型的形容詞
     + 只能放在名詞片語的形容詞
       + 類似限定詞
       + 類似強調語氣的副詞
       + 類似其他的副詞
     + 名詞片語中形容詞的順序
     + 後位修飾的形容詞
       + 複合名詞後面
       + 一些a-開頭的形容詞
       + 外來語與慣用語
   + 補語位置(predicative adjectives)
     + SVOC/SVC
     + 一些a-開頭的形容詞
     + 需要補充的形容詞
       + 不定詞片語
       + 介係詞片語
       + 名詞子句

#### 第5章 副詞
1. 副詞修飾的對象:動詞、形容詞、副詞
2. 副詞的位置:
   + 方法狀態副詞:修飾動詞怎麼作。
     + 儘量靠近要修飾的東西。
     + S + V :可前可後
     + S + V + C :
     + S + V + O :建議在動詞前
     + S + V + O + O :建議在動詞前
     + S + V + O + C
   + 頻率副詞
     + 修飾動作頻率，普通動詞前面，BE動詞與助動詞後面
   + 強調範圍的副詞
     + 修飾詞類不拘
     + 通常前面，例如only
   + 加強語氣的副詞
     + 修飾詞類不拘，例如badly
   + 程度副詞
     + 修飾詞類不拘，如拿掉會影響到語意
   + 連接副詞
     + 與連結詞不同，需要有逗點或分號斷開。
   + 分離副詞
     + 修飾整句或省略，例如theoretically
   + 介副詞
     + 後面沒有副詞，自己當副詞的介系詞。
   + 時間副詞與地方副詞
     + 修飾名詞
     + 修飾動詞
     + 作為補語
3. 副詞種類與位置比較
   + 練習不同例句
4. 形容詞與副詞的判斷
5. 重點
   + 副詞可以修飾的對象一般是動詞、形容詞、副詞,但有幾類副詞也可以修飾名詞。
   + 副詞的位置應盡量與修飾對象接近,而且對象要明確,避免一個飾多個對象都說得通,造成讀者難以確認句意。
   + 依照句型和語意要求,句子中的修飾語可能採用形容詞或副詞,要謹愼判斷。
